# AR Library Navigation 📖  
> “한 걸음, 한 걸음 당신을 목적지까지”



## 프로젝트 소개

AR Library Navigation은 QR 코드 기반 위치 인식과 AR(증강현실) 기술을 결합한  
Android 실내 길 안내 애플리케이션입니다.

본 프로젝트는 Android 환경에서 CameraX를 통한 실시간 카메라 처리,  
Google ML Kit을 활용한 QR 코드 인식,  
Unity + ARCore 기반 AR 오버레이 렌더링을 연동하여 구현되었습니다.

QR 코드를 통해 출발지 정보를 인식하고,  
목적지 데이터를 기반으로 AR 화면에 화살표 형태의 시각적 안내 정보를 오버레이하여  
사용자가 텍스트나 복잡한 지도를 보지 않고도 직관적으로 이동할 수 있도록 설계했습니다.


## 시작 계기

기존 지도 애플리케이션은 텍스트 중심의 안내와 복잡한 UI로 인해  
특히 고령자나 지도 사용이 익숙하지 않은 사용자에게는  
실제 이동 상황에서 활용하기 어렵다는 한계가 있다고 느꼈습니다.

부모님 세대를 포함해,  
“설명을 들어도 혼자서는 길을 찾기 어려운 사용자”를 떠올리며

- 글자 크기 조절이 가능한 UI  
- 사용자가 이해하기 쉬운 화면 구성  
- 텍스트가 아닌 시각적 AR 안내 방식  

을 제공하는 내비게이션을 만들고자 했습니다.

현실 세계를 카메라로 비춘 상태에서 AR 화살표만 따라 이동할 수 있다면,  
휴대폰 화면만 내려다보다가 장애물에 부딪히는 위험을 줄이고  
보다 안전하고 직관적인 이동이 가능할 것이라 판단했습니다.

고령화 시대에 필요한 서비스라고 생각해  
기술적으로 어렵지만 도전하게 된 프로젝트입니다.


## 주요 기능

- QR 코드 기반 위치 인식

- AR 기반 길 안내

- 목적지 데이터 관리

- 경로 계산 설계

> 현재는 ARCore 기반 바닥 평면 인식을 활용한  
기본적인 방향 안내 기능을 중심으로 구현되어 있으며,  
향후 Dijkstra 알고리즘을 적용해  
경로 계산 기능을 확장함으로써 앱의 완성도를 높일 수 있습니다.


## 기술 스택

- Android Studio  
- Unity  
- Kotlin  
- SQLite (Room Database)  
- Jetpack Compose  
- Figma  
- CameraX  
- Google ML Kit API  
- ARCore  
- 외부 3D 모델 라이브러리  
- Dijkstra Algorithm  


## 프로젝트 의의

본 프로젝트는 실제 사용 환경에서 발생하는 문제를 분석하고,  
**문제 정의 → 아이디어 기획 → 설계 → 구현**의 흐름을 거쳐  
실제 앱 출시가 가능한 수준의 완성도 높은 계획으로 완성한 프로젝트입니다.

기획부터 구현까지 전 과정을 경험하며  
서비스 개발의 전체 흐름을 이해할 수 있었고,  
AR·QR과 같이 구현 난이도가 높은 신기술을  
프로젝트에 적용하는 경험을 할 수 있었습니다.

이 과정에서 팀원들과의 협업과 체계적인 설계를 통해,  
새로운 기술 또한 충분히 도전하고 적용할 수 있다는 가능성을 확인할 수 있었습니다.
